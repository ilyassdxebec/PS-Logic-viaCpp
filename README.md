# Problem Solving & Algorithms Journey

This repo documents my learning journey through problem solving, algorithms, data structures, and computer science fundamentals — written in clean, structured **C++**.

## What's Inside

A collection of folders organized by course/level, each containing progressively harder problems covering:

- **Problem Solving** — logical thinking and breaking down problems
- **Algorithms** — sorting, searching, recursion, and more
- **Data Structures** — arrays, matrices, linked lists, and beyond
- **Low Level Concepts** — bits, memory, and how things work under the hood for stronger CS foundations and a better understanding of how computers actually work

## Why C++

C++ forces you to think about memory, performance, and structure. It builds strong fundamentals that make every other language easier to learn afterward.

## Code Style

Every solution is written with clean, readable code — proper function decomposition, meaningful names, and consistent formatting. The goal isn't just to solve problems, it's to solve them well.

## Goal

To build a rock-solid foundation in computer science from the ground up — the right way.
